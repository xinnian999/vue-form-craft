# 滑块（Slider）

## 组件标识

`component: 'Slider'`

## componentProps

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| disabled | `boolean` | 禁用 |
| range | `boolean` | 范围模式 |
| min | `number` | 最小值 |
| max | `number` | 最大值 |
| step | `number` | 步长 |
| onChange | `(value: number \| [number, number]) => void` | 值变化 |
| style | `Record<string, any>` | 自定义样式 |
