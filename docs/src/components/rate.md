# 评分（Rate）

## 组件标识

`component: 'Rate'`

## componentProps

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| disabled | `boolean` | 禁用 |
| max | `number` | 最大分值 |
| allowHalf | `boolean` | 允许半星 |
| showScore | `boolean` | 显示分数 |
| showText | `boolean` | 显示辅助文字 |
| onChange | `(value: number) => void` | 值变化 |
| style | `Record<string, any>` | 自定义样式 |
