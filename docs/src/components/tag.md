# 标签（Tag）

## 组件标识

`component: 'Tag'`

## componentProps

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| text | `string` | 标签内容 |
| type | `'success' \| 'info' \| 'warning' \| 'danger' \| 'primary' \| 'default'` | 类型 |
| closable | `boolean` | 可关闭 |
| size | `'large' \| 'default' \| 'small'` | 尺寸 |
| effect | `'dark' \| 'light' \| 'plain'` | 主题 |
| round | `boolean` | 圆角 |
| color | `string` | 自定义颜色 |
| onClose | `() => void` | 关闭时触发 |
