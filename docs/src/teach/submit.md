# 提交表单

本文介绍了两种提交表单的方式。包括使用内置提交按钮和自定义提交按钮。

## 使用内置提交按钮

《JsonSchema》添加`submitBtn: true`，可开启内置的提交按钮。

点击提交按钮时，会触发表单校验，如果校验通过，会触发`finish`事件。

可以在`finish`事件中，与后端进行交互，处理表单提交。

::: demo

demo/teach/Submit1.vue

:::

## 自定义提交按钮

如果内置的提交按钮无法满足需求，可以手写一个提交按钮。

通过调用实例方法`validate`方法触发表单校验。

校验通过后，与后端进行交互，处理表单提交。

:::
