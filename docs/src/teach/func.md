# 函数传递

如果你的《JsonSchema》需要入库。那么就需要保证是纯json格式，这就代表无法使用函数。

但是我们某些情况下需要给组件传递函数，怎么办？

**答案是：使用[JS表达式](/linkage)。**

**JS表达式支持解析一个函数，这个函数会被返回。**

> 下面是一个例子：定制《日期选择器》只能选择未来的日期。

这种情况下我们可以用**JS表达式**去解析函数的引用。
